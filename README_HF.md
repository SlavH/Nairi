# Nairi Factory

AI Dev Team in a Box — Three AI agents (Architect, Developer, Reviewer) collaborate to build web apps from a single prompt, running on AMD GPUs.

## Quick Start

1. Set `BITNET_BASE_URL` to your AMD GPU endpoint (see docs)
2. Set Supabase env vars for auth
3. Run `npm install && npm run build && npm start`

See [README.md](https://github.com/SlavH/Nairi) for full setup.
